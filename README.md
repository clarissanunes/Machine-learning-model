# Calculadora Python

Este projeto é uma calculadora simples implementada em Python. A calculadora permite ao usuário realizar operações básicas como adição, subtração, multiplicação e divisão.

## Funcionalidades

- Adição
- Subtração
- Multiplicação
- Divisão
